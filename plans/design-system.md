# design System do Portfolio

## 1. Introducao
Esta e uma pagina de autenticacao simples e minimalista

## 2. Fundacoes
- Paleta de cores:
 --text: #eeeeeec;
 --background: #0b0b09
 --primary: #868469;
 --secondary: #4e4d3b;
 --accent: #6e6b4e;

-Tipografia:
 - Titulos: Inter (via google fonts)
 -Texto normal: Sans-serif

-Espacamento
 - Grande (8 ou 16px)

## 3. Componentes
 - Botoes:
   -Todos os botoes vao ter uma sombra pequena na parte inferior
   -Terao arrendondamento de 4px
   -Terao fonte negrito
   -Terao o espacamento definido na Fundacao. 
 - Variacoes:
   -Primario: Seguirao a paleta a risca
   -Secundario: Tera o valor invertido do fundo/texto
 -Campo de texto (input)
   - Todos os inputs vao:
     -Terao arredondamento de 4px
     -Seguirao espacamento definido na fundacao 

## 4. Guia de Uso
- No CSS sempre usar variaveis para cores e elementos da fundacao ao
 inves de espalhar pelo codigo.
- Manter a pagina sempre com foco principal em ter os elementos centralizados.
