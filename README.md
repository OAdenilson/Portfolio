# Portfolio Website

Portfólio pessoal desenvolvido para apresentar minha trajetória, habilidades e projetos na área de tecnologia.

## Sobre o Projeto

Este projeto foi desenvolvido utilizando HTML e CSS a partir de uma abordagem baseada em documentação e especificação de requisitos.

Antes da implementação foram criados os documentos:

- design-system.md
- spec-geral.md

Esses documentos serviram como base para definir a estrutura, identidade visual e comportamento da aplicação.

Além do desenvolvimento, o projeto também foi utilizado como prática de planejamento, organização e utilização de Inteligência Artificial como ferramenta de apoio ao processo de criação.

## Tecnologias Utilizadas

- HTML5
- CSS3
- Git
- GitHub
